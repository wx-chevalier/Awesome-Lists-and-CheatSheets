# Chameleon

# Alita

- [2019-深入剖析 Alita 引擎：解密如何实现 RN 转微信小程序](https://www.infoq.cn/article/eoGSdMk_VOjs2Rafjr37): 我们对Alita的期望是不能对JSX语法有太多限制，不能有侵入性，不给React Native的开发者带来太多的负担。所以最终Alita`引擎没有基于任何现有的编译时方案，而是另辟溪路，走了一条颇具开创性的运行期处理方案。
